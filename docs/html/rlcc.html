rlcc¶

RL Common Components

The rlcc package contains data structures and algorithms for common components of reinforcement learning.

The package uses pytorch for underlying tensor operations.

rlcc.transition(*args)[source]¶

Creates a transition object from arguments.

Parameters:args (tensors) – tuple of (state, action, reward, next_state, is_terminal) tensors
rlcc.tensor(data)[source]¶

Constructs a torch tensor with data that returns a tensor of float data type.

Parameters:data (Can be a list, tuple, NumPy ndarray, scalar, and other types.) – data for the tensor.
class rlcc.Transition(state, action, reward, next_state, is_terminal)¶
action¶

Alias for field number 1: The action taken in the transition.

is_terminal¶

Alias for field number 4: If terminal then 1.0, 0.0 otherwise.

next_state¶

Alias for field number 3: The state the transition is to.

reward¶

Alias for field number 2: The reward for transitioning.

state¶

Alias for field number 0: The state the transition is from.
